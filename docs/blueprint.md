# **App Name**: Vibee OS Lite

## Core Features:

- Vibe Posting: Users input their current feeling with text, an emoji is selected (AI-assisted or manual), and a background color is assigned based on the emotion. A sentiment analysis tool is employed to suggest appropriate emojis.
- Emotion Feed: Users can explore feeds based on specific emotions, filtering by tags to see posts from others feeling the same.
- Vibe Matching: The application matches users posting similar vibes within a recent timeframe, allowing them to initiate a chat.
- Mood History: A personal chart displays the user's weekly emotional flow to promote self-reflection.
- Anonymous Posting: Users have the option to post anonymously to encourage honesty and safety.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a calm and reflective mood.
- Background color: Very light desaturated Indigo (#F0F2F9) to provide a subtle backdrop.
- Accent color: Blue (#5677FC) as a vibrant and energetic color to make the UI elements pop.
- Font: 'Inter', a sans-serif font suitable for both headlines and body text due to its modern and neutral appearance.
- Large, expressive emojis to convey emotions, complemented by minimalistic vector icons for navigation and actions.
- A clean and minimal layout with ample spacing, focusing on content clarity and ease of use. Emotion-based color bubbles for posts.
- Smooth transitions and subtle animations to enhance the user experience, particularly when transitioning between emotion zones and during vibe matching.
